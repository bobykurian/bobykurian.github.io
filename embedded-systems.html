Embedded Systems

Preamble

This course aims to design an embedded electronic circuit and implement the same.

Prerequisite

ECT 203 Logic Circuit Design, ECT 202 Analog Circuits, ECT 206 Computer Architecture and Microcontrollers.

Course Objectives

  • To introduce the fundamentals of embedded systems and familiarize students with embedded electronic circuit design methodologies.
  • To develop the ability to understand microcontroller architectures, interfaces, and programming required for embedded system development.
  • To enable students to design, implement, and test embedded electronic solutions for real-world engineering applications.

Syllabus

Module 1: Introduction to Embedded Systems

1.1 Complex Systems and Microprocessors:
Embedding Computers, Characteristics of Embedded Computing Applications, Application of Microprocessors, The Physics of Software, Challenges in Embedded Computing System, Characteristics and quality attributes of an embedded system, Performance in Embedded Computing.

1.2 The Embedded System Design Process:
Requirements, Specification, Architecture Design, Designing Hardware and Software Components, System Integration.

1.3 Formalisms for System Design:
Structural Description, Behavioral Description, An embedded system design example.

1.4 Embedded Product Development Cycle (EDLC):
Different phases of EDLC, EDLC models.

Module 2: Embedded System Interfacing and Peripherals

2.1 Communication Devices:
UART, HDLC, SPI, I2C, CAN, USB, ISA, PCI, PCI-X Standards.

2.2 Memory:
Flash, EEPROM, SRAM, DRAM, Cache, Memory Mapping, Addresses, MMU, DMA.

2.3 I/O Device and Interrupts:
Interrupt sources, recognizing interrupts, ISR, Device drivers, shared data problem, interrupt latency.

Module 3: ARM Processor Fundamentals

3.1 ARM Processor Architecture:
Acorn RISC Machine, Architectural inheritance, ARM programmer's model, development tools.

3.2 ARM Assembly Language Programming:
Data processing instructions, data transfer instructions, control flow instructions, writing simple programs.

3.3 ARM Organization and Implementation:
3-stage and 5-stage pipeline, instruction execution, ARM implementation, coprocessor interface.

Module 4: ARM Programming

4.1 Architectural Support for High-Level Languages:
Data types, floating-point architecture, expressions, loops, memory use, runtime environment.

4.2 The Thumb Instruction Set:
CPSR Thumb bit, Thumb programmer's model, branch instructions, data processing, register transfer, multiple transfer, breakpoint instruction, applications.

4.3 Architectural Support for System Development:
ARM memory interface, AMBA.

4.4 Programming:
Assembly and C language programming for embedded systems.

Module 5: Real-Time Operating Systems (RTOS)

5.1 OS Basics:
Kernel, types of operating systems.

5.2 Real-Time Operating Systems:
Tasks, processes, threads, scheduling, multitasking, communication, synchronization, device drivers, choosing an RTOS.
